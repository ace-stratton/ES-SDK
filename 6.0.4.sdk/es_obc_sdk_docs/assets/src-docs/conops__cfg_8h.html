◆ CONOPS_3AXIS_CONTROL_ANG_VEL_SAFETY_THRESH

#define CONOPS_3AXIS_CONTROL_ANG_VEL_SAFETY_THRESH   ((int32_t)39)

Single axis angular velocity safety threshold allowing the activation of 3-Axis control modes.

The value defines the following range [-39, +39] mrad/s. If the estimators indicate that the angular velocity for each axis is within this range, then the activation of a 3-Axis (or Y-Momentum) mode is safe.

Note that this value depends on the used reaction wheel, the current value is indicative for medium wheels.
