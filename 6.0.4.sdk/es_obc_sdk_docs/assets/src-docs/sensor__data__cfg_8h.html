Get sensor data.

Parameters
[in]sensor_data_sensors_tsens_type - sensor type
[in]sds_all_sens_data_t* const p_sens_data - data in own coordinate frame
[in]sds_all_sens_data_sat_step_t* const p_sens_data_sat - data in sat coordinate frame
[in]constvoid **p_sens_data_mem - pointer to pointer used for the DC update
[in]constvoid **p_sens_data_sat_mem - pointer to pointer used for the DC update
Returns
sds_op_status_t
Note
This function makes the correct calls to get the sensor data in its own coordinate frame and updates it in the relevant data structure. It calls the data conversion to satellite coordinate frame and updates the current and old entries in the sat data structure
