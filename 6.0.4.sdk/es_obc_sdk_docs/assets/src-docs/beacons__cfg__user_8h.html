◆ beacons_cfg_get_active_preset()

uint8_t beacons_cfg_get_active_preset ( void  )

Determines the active beacon preset id to use for the selection of beacon messages.

The beacon configuration provides a set of beacon preset identifiers which are coupled with a set of beacon messages to transmit. In this case, the preset id is determined from the OBC operational mode.

Returns
ID of the active beacon preset

The function is used to simulate a change in the active beacon preset. Its effect will remain until next reboot or until the beacons_cfg_restore_active_preset is gets called.
