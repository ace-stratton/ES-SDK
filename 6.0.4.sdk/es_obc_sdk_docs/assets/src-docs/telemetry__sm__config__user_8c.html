Detailed Description

This file contains the Telemetry SM hook functions (e.g. on_entry/on_exit/actions and guards).

The functions here are directly called by the libhsm engine based on the generated code from telemetry_sm.yaml.

Warning
Pay attention to edit this file only between the USER_CODE_START and USER_CODE_END comment blocks in order to ensure that the auto-merge function of the libhsm generator will preserve your changes!
