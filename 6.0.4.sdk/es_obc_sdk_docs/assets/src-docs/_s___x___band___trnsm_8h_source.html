61/*---------------------------------------------------------------------------------------
62Message 0x0049 - Tranciver Chip AD9364 All Tranciver parameters.
63-----------------------------------------------------------------------------------------
64 Description: Message to XTR to write data to the AD9364
65 Total payload: 12 bytes
66 Structure of the payload: ALL TRANCIVER PARAMS
67 Byte 0 | 1 byte | Symbol_Rate
68 Byte 1 | 1 byte | TX_Power
69 Byte 2 | 1 byte | Modulation_Code
70 Byte 3 | 1 byte | Roll_Off
71 Byte 4 | 1 byte | Pilot_Signal_On_Off

73 Byte 6 | 2 byte | Pre_Trans_Staff_Delay
74 Byte 8 | 4 byte | Central_Frequency
75*/
