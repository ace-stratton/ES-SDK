Detailed Description

Driver for the MAC Layer of the ESPS I over RS-485.

Usage:

  • Populate the configuration fs_MacConfig in esps_drv_config.c
  • Pass it to the interface state fs_asMACStates in esps_drv_config.c
  • Send messages to the network by calling MAC_DrvEnqMessage, configured in in ESSA_StackConfig.c
  • The received messages will be dispatched when MAC_DISP_vDispatchMessages is called in esps_drv_dispatcher_thread.c
