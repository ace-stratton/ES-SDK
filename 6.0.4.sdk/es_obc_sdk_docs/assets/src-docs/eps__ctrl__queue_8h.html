Searches for a specific request in the queue.

Parameters
chThe searched element must have this as its requested channel
flagThe searched element must have this as its wait flag
Returns
Pointer to the queue element that matches the search criteria
Warning
Although the queue access is thread-safe, this function returns a pointer, which in itself is not. Additional synchronization should be employed if the function is used
