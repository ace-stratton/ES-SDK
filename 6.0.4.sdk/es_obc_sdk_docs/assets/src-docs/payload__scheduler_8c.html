Adds a new scheduling record to the schedule table (if there are free slots available).

Parameters
[in]payload_idID of the payload for which to add a scheduling entry
[in]timeUNIX-timestamp specifying when the payload shall be started
[in]durationduration in seconds before the payload is stopped by the scheduler
[in]adcs_dataADCS pointing control parameters
[in]flagspayload-specific flags interpreted by the payload handler
[in]pay_argsa pointer to a buffer keeping payload-specific arguments (could be a configuration file name or just a data structure interpreted by the payload handler
Returns
true: the scheduling record was successfully added, otherwise false

void PayloadScheduler_Reload ( const char *  p_fname)

Triggers a reload of the scheduling data from the SD card.

The actual reload is performed in the OS task of the payload scheduler. The function just sets a flag to indicate the operation is requested externally.

Parameters
[in]p_fnameA pointer to the schedule file name to load
