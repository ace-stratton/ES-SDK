◆ sys_printf_char_func_t

typedef void(* sys_printf_char_func_t) (char ch)

A pointer to a function which implements printing a single character to an output sink.

Parameters
chThe ASCII code of the character to print
Note
The function is provided by the user. Different implementations can be used to print to different output sinks (e.g. UART, file, ARM trace port, etc.). The function returns control when the data has been printed.
