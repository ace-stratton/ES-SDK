Detailed Description

Optional extended bundle footer which contains information about the file name to use for the uploaded buffer.

The structure extends the current FWUPD protocol handling to support regular file uploads with a file name specified by the user. At the same time, the protocol remains compatible to the existing tooling because the additional footer is actually part of the payload data and will only be used if formatted as expected and its CRC is calculated correctly.
