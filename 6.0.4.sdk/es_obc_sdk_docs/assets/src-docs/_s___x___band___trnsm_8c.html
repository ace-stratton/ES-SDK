Request a command from ESTTC gateway.

Enqueue an SXBand command for execution.

Parameters
[input]Identifier - SXBand module identifier to send the command to
[input]Cmd - Command number that is requested to be executed
  • 0x01 Symbol Rate
  • 0x02 Tx Power
  • 0x03 Central Frequency
  • 0x04 MODCOD
  • 0x05 Roll-Off
  • 0x06 Pilot Signal On/Off
  • 0x07 FEC Frame size
  • 0x08 Pretransmission Staffing Delay
  • 0x09 Change All Parameters
  • 0x0A Simple Report
  • 0x0B Full Report
  • 0x0C Change Baudrate
  • 0x30 TxMode
  • 0x31 Load Mode
  • 0x32 Prepare to ShutDown
  • 0x35 FW update
  • 0x40 Dir
  • 0x50 Delete
  • 0x55 Delete all
  • 0x60 Upload a file
  • 0x61 Download a file
  • 0x70 Send a file
[input]Type - accepts only: 'R' - Read command, 'W' - Write commands
[input]size - size of the input data if used (if pData == NULL, this parameter is ignored)
[input]pData - pointer to the input data (if not needed set to NULL. This can be settings, file name etc.)
[input]pOnCmdResponse - pointer to a user-defined callback which receives the result from the command
[output]p_cmd_eta_ms - milliseconds to wait for the command to finish
Returns
none
