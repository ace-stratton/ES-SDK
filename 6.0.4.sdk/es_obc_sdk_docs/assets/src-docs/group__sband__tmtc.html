Detailed Description

Interact with the Endurosat S Band TMTC module. This module is connected to the OBC using the RS-485 bus and the ESPS I protocol.

Usage

OBC monitors that the SBand TMTC is active by requesting its uptime through the ESPS I bus. using the FP function SBAND_TMTC_ReadUpTimeReq.

Whenever a response arrives, the callback from the ESPS FP is called (sband_tmtc_uptime_response_handler) and the status of the SBand can be checked by calling sband_tmtc_is_active.

FP commands

This module enables access to all the Function Protocol commands generated for the S Band TMTC
