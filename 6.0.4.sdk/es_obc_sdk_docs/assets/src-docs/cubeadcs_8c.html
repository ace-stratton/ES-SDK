◆ CubeADCS_SendCmdWithChecks()

eCubeADCS_Comm_Status_t CubeADCS_SendCmdWithChecks ( CubeADCS_CmdCfg_t *  sCmd,
uint8_t  u8Update,
uint8_t  u8Valid 
)

Wrapper around CubeADCS_SendCmd used to correctly update the operational status of the CubeADCS.

Parameters
[in]sCmdpointer to command details (context)
[in]u8Updatevalue to indicate whether an update of the diagnostic counters is requested or not
[in]u8Validvalue to indicate whether the update should happen only on validated data
Returns
status of the command request
Note
This wrapper updates the operational status structure on request and if command request resulted in valid data.

Data is read first into a temporary location and if this data is found to be valid (by expecting OK from the communication and a valid first byte) it is written in the operational status structure.
