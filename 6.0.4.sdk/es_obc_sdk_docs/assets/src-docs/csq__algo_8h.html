◆ csq_algo_policy_t

enum csq_algo_policy_t

Defines the types of overwrite policy supported by the algorithm.

Enumerator
CSQ_ALGO_POLICY_NO_OVERWRITE 

does not overwrite older entries when the queue gets full

CSQ_ALGO_POLICY_OVERWRITE 

overwrites oldest unprocessed entries if the queue gets full

CSQ_ALGO_POLICY_MAX 

used for range checking only
