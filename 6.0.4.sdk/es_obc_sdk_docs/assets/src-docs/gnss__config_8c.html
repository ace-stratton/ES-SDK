void gnss_config_tm_callback ( const gnss_time_t *  t)

This callback is invoked on every new reception of time logs.

must be implemented by the user

Note
Make sure to not put heavy operations inside the callback as this might lead to missed log events.
The data is validated only for its format and CRC. In order to make sure that the time itself is correct the user can use gnss_is_tm_valid()

[in]timParsed time log
