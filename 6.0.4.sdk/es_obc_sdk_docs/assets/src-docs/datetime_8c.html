◆ datetime_decrement_timestamp()

bool datetime_decrement_timestamp ( datetime_date_t *  p_date,
datetime_time_t *  p_time 
)

decrements a datetime_time_t structure with one second if underflow happens (ie 00:00:00 => 23:59:59), decrement datetime_date_t with one day

Parameters
[in,out]timepointer to a datetime_time_t structure, which will be decremented
[in,out]datepointer to a datetime_date_t structure, which will be decremented
Returns
true on success. false on error. Error may be due to NULL pointers, invalid time/date or underflow (going below the year DATETIME_YEAR_MIN). On error p_date and p_time remain unchanged.
