◆ MCU_InitSdCard()

uint8_t MCU_InitSdCard ( uint8_t  cards,
uint8_t  cardHwReset 
)

Reset SD card and initialise all components needed to use it.

Parameters
[input]cards:
  • 1 - SD card 1
  • 2 - SD card 2
  • 3 - Both SD cards 1 and 2
[input]cardHwReset:
  • pdTRUE - do a HW reset of the SD card (power off, delay, power on)
  • pdFALSE - reset only the SDIO and FAT file system on the MCU
[output]none
Returns
uint8_t:
  • 1 - SD card 1 Successful
  • 2 - SD card 2 Successful
  • 3 - SD card 1 and 2 Successful
  • dpFALSE- Fail
Note
none
