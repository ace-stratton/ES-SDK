◆ eps_ctrl_unblock_user_thread()

void eps_ctrl_unblock_user_thread ( void  )

Unblocks the thread which has issued the currently processed request.

Note
If the notification has been cancelled (i.e. the requesting thread is no longer waiting) no event will be set
Warning
This method is designed to be called from the eps_ctrl state machine when processing of the current request has finished. Do not attempt calling it from a different context!
