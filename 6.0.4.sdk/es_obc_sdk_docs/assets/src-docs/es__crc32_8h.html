Calculate CRC32 checksum over given data length with a parameter for init value of the checksum.

Parameters
initinitial value for the remainder
ptrpointer to data first byte
sizesize of data to calculate the checksum for
Returns
checksum value
Note
Used polynomial is 0x04C11DB7. Implementation is standard table algorithm with reflection and byte reversal. Implementation details borrowed from literature.
