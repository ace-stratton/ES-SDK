Enumeration Type Documentation

◆ trace_level_t

enum trace_level_t

Severity level of a trace entry

Enumerator
TRACE_LVL_INFO 

used to trace important non-anomalous events, e.g. initialization stages, start of a service, etc.

TRACE_LVL_WARN 

used to trace anomalies that prevents proper operation, but is recoverable

TRACE_LVL_ERROR 

used to trace anomalies that are fatal to a functionality, but not to the system as a whole

TRACE_LVL_FATAL 

used to trace anomalies that are fatal to the system and require reset

TRACE_LVL_DEBUG 

used to output detailed information for testing and investigation purposes

TRACE_LVL_COUNT 

number of severity levels, used for internal checks
