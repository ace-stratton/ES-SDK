Detailed Description

Beacons sink public interface.

The interface is intended to be implemented by beacon sink adapters who take care of beacon data processing. The beacon service uses that interface to feed beacon data to the sink but it remains ignorant of what happens to the data in the end. It is the responsibility of the adapter to handle how the data is actually processed.
